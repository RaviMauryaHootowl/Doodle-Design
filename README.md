## Install

    $ git clone https://github.com/RaviMauryaHootowl/Doodle-Design.git
    $ cd PROJECT
    $ npm install
    $ cd client
    $ npm install

## Run in Dev Mode

    $ npm run dev

## Run only Client

    $ npm run client
  
## Run only Server

    $ npm run server
